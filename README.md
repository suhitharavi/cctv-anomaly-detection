# Real-Time CCTV Anomaly Detection

A **hybrid classical–quantum** anomaly detection system for CCTV video: classical deep learning with a PennyLane quantum layer, trained on the UCF-Crime dataset.

### Install
### backend

cd backend

py -3.10 -m venv venv

venv\Scripts\activate 

pip install -r requirements.txt

python app.py

### frontend
cd frontend

npm install

npm run dev
